# braintapper-svelte-datepicker

Datepicker control for Svelte


# Usage

## Component Import

Import the components you need in your <script> tag:

```
  import Datepicker from "braintapper-svelte-datepicker";
```

## Example Markup

```
  <DatePicker/>
```
